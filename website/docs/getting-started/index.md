# Getting Started

Learn how to install, configure, and start using GoQueue.

---

## Overview

Getting started with GoQueue is straightforward:

1. **Install** - Download the binary or use Docker
2. **Configure** - Set up your broker configuration
3. **Run** - Start the broker and create topics
4. **Integrate** - Use the REST API, gRPC, or Go client

## Quick Links

- [Installation Guide](installation) - Install GoQueue on your system
- [Quickstart Tutorial](quickstart) - Get up and running in 5 minutes
- [Kubernetes Deployment](kubernetes) - Deploy to Kubernetes

## Prerequisites

- **Go 1.21+** (for building from source)
- **Docker** (for containerized deployment)
- **~100MB disk space** for the binary
- **64-bit Linux, macOS, or Windows**
